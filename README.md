# MSG Portfolio

Portfolio professionnel moderne et responsive développé avec React, TypeScript et Tailwind CSS.

![Portfolio Preview](./public/preview.png)

## 🚀 Fonctionnalités

- **Design moderne** : Interface élégante avec animations fluides
- **Thème sombre/clair** : Basculement automatique selon les préférences système
- **Responsive** : Optimisé pour tous les appareils (mobile, tablette, desktop)
- **Animations** : Animations Framer Motion pour une expérience utilisateur fluide
- **Formulaire de contact** : Envoi direct d'emails via Formspree
- **SEO optimisé** : Métadonnées complètes pour les moteurs de recherche
- **Performance** : Code optimisé et chargement rapide

## 🛠️ Technologies utilisées

### Frontend
- **React 18** - Bibliothèque JavaScript pour interfaces utilisateur
- **TypeScript** - JavaScript typé pour plus de robustesse
- **Vite** - Outil de build ultra-rapide
- **Tailwind CSS** - Framework CSS utilitaire
- **Framer Motion** - Bibliothèque d'animations

### UI/UX
- **Radix UI** - Composants accessibles et non-stylisés
- **Lucide React** - Icônes modernes et cohérentes
- **shadcn/ui** - Composants UI hautement personnalisables

### Outils de développement
- **ESLint** - Linting du code JavaScript/TypeScript
- **Vitest** - Framework de tests unitaires
- **React Hook Form** - Gestion des formulaires
- **Zod** - Validation des données
